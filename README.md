# FamilyFinance PRO 💰

אפליקציית ווב לניהול תקציב משפחתי — גרסה קלה, מהירה ועמידה ללא שרת, ללא התקנה וללא תלות בספריות חיצוניות. כל המידע נשמר מקומית בדפדפן באמצעות localStorage.

## איך מפעילים

1. פותחים את [index.html](index.html) בכל דפדפן.
2. אפשר גם לפתוח אותו ישירות מהתיקייה או לגרור את הקובץ לחלון דפדפן.
3. אין צורך ב-build, בהתקנת חבילות או בחיבור לאינטרנט.

> הערה: הנתונים נשמרים רק במכשיר ובדפדפן שבו נעשה השימוש. אם מוחקים נתוני דפדפן או localStorage, המידע יימחק.

## מה האפליקציה עושה

### דשבורד דינמי
בראש המסך מוצגת "יתרה פנויה חודשית" יחד עם אריחי סיכום לכל קטגוריה קיימת במערכת. כל שינוי בתנועות מעדכן את המספרים באופן מיידי.

### סוגי תנועות נתמכים

| סוג | תיאור |
|---|---|
| הכנסה | סכום קבוע שמתווסף לתזרים הפנוי |
| הוצאה קבועה | הוצאה חוזרת עם תדירות חודשית או שנתית |
| תשלומים שונים | עסקה עם מספר תשלומים ותאריך התחלה, עם חישוב יתרה שנותרה |
| הלוואה | גרסה של תשלומים שונים עם ריבית ויום חיוב בחודש |
| חיוב חד-פעמי | תנועה עם שם, סכום ותאריך, למשל חיוב בכרטיס אשראי |

### ניהול קטגוריות
במסך "ניהול קטגוריות" אפשר:
- לשנות את שמות ארבע הקטגוריות המובנות
- ליצור קטגוריות מותאמות אישית
- לבחור סוג התנהגות כלכלית לכל קטגוריה חדשה
- להקצות אימוג'י אוטומטי לפי שם הקטגוריה
- לשנות שם של קטגוריה מותאמת אישית קיימת
- למחוק קטגוריה מותאמת אישית, כולל כל התנועות המשויכות אליה לאחר אישור

### ניהול תנועות
- הוספת תנועה חדשה עם טופס דינמי שמשתנה לפי סוג התנועה
- עריכה ישירה מתוך הכרטיסים
- ארכוב של תנועות בלי למחוק אותן לצמיתות
- מחיקה סופית רק מתוך הארכיון
- מעבר בין "תנועות פעילות" ו"ארכיון המערכת"

### תחזית חודשית
האפליקציה כוללת גם תחזית יתרה פנויה לשישה חודשים קדימה, עם תצוגת גרף וטבלה.

## מבנה טכני

- אפליקציה עם קובץ יחיד: HTML, CSS ו-JavaScript ללא ספריות חיצוניות
- אחסון מקומי באמצעות localStorage
- תמיכה ב-RTL ועברית כברירת מחדל

## רעיונות לשיפור עתידי

- סנכרון ענן בין מכשירים
- ייצוא/יבוא של נתונים
- דוחות חודשיים וגרפים נוספים
- הגדרות נוספות למעקב אחר מטרות תקציביות
